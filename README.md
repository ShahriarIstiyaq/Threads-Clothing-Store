
# Threads Clothing Store

A visually appealing, responsive e-commerce website for a modern clothing brand. This single-page application allows users to browse products, manage their shopping cart, and experience a seamless online shopping journey. Built with HTML, CSS, and JavaScript, the project emphasizes clean UI/UX and interactive features.

## Features

- **Responsive Navbar:** Easy navigation across all sections and devices.
- **Hero & Banner Sections:** Engaging promotional banners and hero images for brand highlights.
- **Dynamic Product Catalog:** Products loaded from JSON, with filtering and featured/new arrivals.
- **Interactive Shopping Cart:** Add, remove, and update product quantities with persistent cart storage (localStorage).
- **Coupon/Discount System:** Apply discount codes at checkout with real-time cart total updates.
- **Customer Reviews:** Showcases testimonials and ratings for social proof.
- **Slideshow Carousel:** Interactive slider for exclusive collections and promotions.
- **Newsletter Signup:** Integrated email capture for marketing.
- **Real-time Notifications:** Instant feedback for cart actions and coupon application.
- **Modern, Responsive Design:** Optimized for all devices using HTML5, CSS3, and vanilla JavaScript.

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- JSON (for product data)

## Getting Started

1. **Clone the repository:**
   ```powershell
   git clone https://github.com/istiyaq13/Threads-Clothing-Store.git
   ```
2. **Navigate to the project directory:**
   ```powershell
   cd "Threads Clothing Store"
   ```
3. **Open `index.html` in your browser.**

> No build tools or frameworks required—everything runs in the browser.
